## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **Live GitHub Data** | Fetches real-time stats from the GitHub API |
| 🎨 **Dark/Light Theme** | Seamless theme switching with CSS variables |
| 📈 **3 Interactive Charts** | Language distribution, contribution heatmap, code frequency |
| 📌 **Pinned Repositories** | Showcases your top 6 repos by stars |
| 🔄 **Drag & Drop** | Rearrange stats cards to personalize your dashboard |
| 📥 **Export Markdown** | Generate a polished README.md with your stats |
| 🖼️ **Export PNG** | Capture the dashboard as a high-res image |
| 🌐 **Social Sharing** | Share on Twitter, LinkedIn, WhatsApp, Reddit |
| ⏱️ **Wakatime Integration** | Simulated coding activity metrics |
| 📱 **Fully Responsive** | Works on all devices from mobile to desktop |
| 🤝 **Apache 2.0 Licensed** | Open source with Code of Conduct |
